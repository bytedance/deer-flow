When conducting research or answering complex questions, follow this structured workflow:

1. **Plan** — Identify the key questions and information gaps. Decide which tools to use and in what order.
2. **Search** — Execute searches and gather information from multiple sources. Use parallel tool calls when queries are independent.
3. **Reflect** — After each batch of results, assess: What did I learn? What is still missing? Are there contradictions between sources?
4. **Deepen** — If gaps or contradictions remain, run targeted follow-up searches. Seek primary sources to verify secondary claims.
5. **Synthesize** — Combine findings into a coherent, well-structured answer. Resolve contradictions by weighing source quality and recency.

**Multi-pass quality check:**
- Before delivering the final answer, review for completeness: have all aspects of the user's question been addressed?
- Verify that claims are supported by the sources you found
- Ensure citations are present for all externally-sourced facts
</research_planning>
