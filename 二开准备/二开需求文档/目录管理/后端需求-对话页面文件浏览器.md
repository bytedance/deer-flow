# 后端需求文档：对话页面文件浏览器 API

> 目标：为前端文件浏览器提供文件读写和管理的后端 API 支持

---

## 1. 需求概述

### 1.1 功能目标
为前端对话页面的文件浏览器功能提供后端 API 支持，包括：
- 读取 AI 工作目录下的文件内容
- 写入/保存文件内容
- 删除文件
- 浏览目录结构（支持文件类型过滤）

### 1.2 工作目录说明
- 每个线程（thread）有独立的工作目录
- 虚拟路径：`/mnt/user-data/workspace/{thread_id}/`
- 实际路径由系统配置决定（通过 `thread_data['workspace_path']` 获取）

---

## 2. 技术栈

| 技术 | 用途 |
|------|------|
| Python 3.x | 后端语言 |
| FastAPI | Web 框架 |
| LangGraph SDK | AI 代理框架 |
| 原生 os/pathlib 模块 | 文件系统操作 |

## 4. 新增 API 接口

### 4.1 接口清单

| 方法 | 路径 | 功能 | 优先级 |
|------|------|------|--------|
| GET | `/api/filesystem/browse` | 浏览目录 | 高 |
| GET | `/api/filesystem/read` | 读取文件内容 | 高 |
| POST | `/api/filesystem/write` | 写入文件内容 | 高 |
| DELETE | `/api/filesystem/delete` | 删除文件 | 高 |

### 4.2 通用说明

**所有接口都需要：**
- `thread_id` 参数（URL query 或 header）
- 路径使用虚拟路径格式（`/mnt/user-data/workspace/...`）
- 进行路径验证，防止路径遍历

**认证方式：**
- 使用现有的线程认证机制
- 确保用户只能访问自己线程的工作目录

## 5. 接口详细设计

### 5.1 浏览目录

```python
# GET /api/filesystem/browse?thread_id={thread_id}&path={virtualPath}&extensions=md,txt,json,log

# Request 参数
thread_id: str          # 线程 ID（必需）
path: str              # 虚拟目录路径（可选，默认工作目录）
extensions: str        # 文件扩展名过滤，逗号分隔（可选）

# Response Success (200)
{
    "currentPath": "/mnt/user-data/workspace/subfolder",
    "parentPath": "/mnt/user-data/workspace",
    "entries": [
        {
            "name": "folder1",
            "path": "/mnt/user-data/workspace/subfolder/folder1",
            "isDirectory": true
        },
        {
            "name": "notes.md",
            "path": "/mnt/user-data/workspace/subfolder/notes.md",
            "isDirectory": false
        }
    ]
}

# Response Error (400)
{
    "error": "Missing thread_id parameter"
}

# Response Error (403)
{
    "error": "Access denied: Invalid path"
}
```

**实现要点：**
```python
async def handle_browse(
    thread_id: str,
    path: str = None,
    extensions: str = None
):
    # 1. 获取线程数据
    thread_data = get_thread_data(thread_id)
    workspace_path = thread_data.get("workspace_path")
    
    # 2. 确定要浏览的目录
    browse_path = path or f"{VIRTUAL_PATH_PREFIX}/workspace"
    
    # 3. 验证路径
    validate_local_tool_path(browse_path, thread_data, read_only=True)
    
    # 4. 解析为实际路径
    actual_path = _resolve_and_validate_user_data_path(browse_path, thread_data)
    
    # 5. 列出目录
    entries = list_directory(actual_path)
    
    # 6. 过滤文件（如果提供了 extensions）
    if extensions:
        allowed_exts = [ext.strip().lower() for ext in extensions.split(',')]
        entries = [
            e for e in entries
            if e['isDirectory'] or 
            e['name'].lower().endswith(tuple(allowed_exts))
        ]
    
    # 7. 转换回虚拟路径
    for entry in entries:
        entry['path'] = actual_to_virtual_path(entry['path'], thread_data)
    
    return {"currentPath": browse_path, "entries": entries}
```

**实现要点：**
```python
async def handle_read_file(
    thread_id: str,
    path: str,
    start_line: int = None,
    end_line: int = None
):
    # 1. 验证参数
    if not path:
        return {"error": "Missing path parameter"}, 400
    
    # 2. 获取线程数据
    thread_data = get_thread_data(thread_id)
    
    # 3. 验证路径
    validate_local_tool_path(path, thread_data, read_only=True)
    
    # 4. 解析为实际路径
    actual_path = _resolve_and_validate_user_data_path(path, thread_data)
    
    # 5. 检查文件是否存在
    if not os.path.isfile(actual_path):
        return {"error": "File not found"}, 404
    
    # 6. 检查文件大小（限制最大 10MB）
    file_size = os.path.getsize(actual_path)
    MAX_FILE_SIZE = 10 * 1024 * 1024
    if file_size > MAX_FILE_SIZE:
        return {"error": "File too large"}, 400
    
    # 7. 读取文件
    with open(actual_path, 'r', encoding='utf-8') as f:
        content = f.read()
    
    # 8. 如果指定了行范围，截取内容
    if start_line is not None and end_line is not None:
        lines = content.splitlines()
        content = '\n'.join(lines[start_line-1:end_line])
    
    # 9. 返回结果
    stat = os.stat(actual_path)
    return {
        "path": path,  # 返回虚拟路径
        "name": os.path.basename(actual_path),
        "content": content,
        "size": file_size,
        "modifiedAt": datetime.fromtimestamp(stat.st_mtime).isoformat()
    }
```

```python
async def handle_write_file(
    thread_id: str,
    path: str,
    content: str,
    append: bool = False
):
    # 1. 验证参数
    if not path or content is None:
        return {"error": "Missing path or content"}, 400
    
    # 2. 获取线程数据
    thread_data = get_thread_data(thread_id)
    
    # 3. 验证路径
    validate_local_tool_path(path, thread_data, read_only=False)
    
    # 4. 解析为实际路径
    actual_path = _resolve_and_validate_user_data_path(path, thread_data)
    
    # 5. 确保父目录存在
    os.makedirs(os.path.dirname(actual_path), exist_ok=True)
    
    # 6. 写入文件
    mode = 'a' if append else 'w'
    with open(actual_path, mode, encoding='utf-8') as f:
        f.write(content)
    
    # 7. 返回结果
    stat = os.stat(actual_path)
    return {
        "success": True,
        "path": path,
        "size": stat.st_size,
        "modifiedAt": datetime.fromtimestamp(stat.st_mtime).isoformat()
    }
```

```python
async def handle_delete_file(
    thread_id: str,
    path: str
):
    # 1. 验证参数
    if not path:
        return {"error": "Missing path parameter"}, 400
    
    # 2. 获取线程数据
    thread_data = get_thread_data(thread_id)
    
    # 3. 验证路径
    validate_local_tool_path(path, thread_data, read_only=False)
    
    # 4. 解析为实际路径
    actual_path = _resolve_and_validate_user_data_path(path, thread_data)
    
    # 5. 检查是否为文件（不能删除目录）
    if not os.path.isfile(actual_path):
        return {"error": "Can only delete files, not directories"}, 403
    
    # 6. 删除文件
    os.remove(actual_path)
    
    # 7. 返回结果
    return {
        "success": True,
        "path": path
    }
```

## 6. 路由实现

### 6.1 FastAPI 路由注册

```python
# backend/packages/harness/deerflow/api/filesystem.py

from fastapi import APIRouter, HTTPException, Query
from pydantic import BaseModel
from typing import Optional

router = APIRouter(prefix="/api/filesystem", tags=["filesystem"])

class WriteFileRequest(BaseModel):
    thread_id: str
    path: str
    content: str
    append: bool = False

@router.get("/browse")
async def browse_directory(
    thread_id: str = Query(..., description="Thread ID"),
    path: Optional[str] = Query(None, description="Virtual directory path"),
    extensions: Optional[str] = Query(None, description="File extensions filter (comma-separated)")
):
    """Browse directory contents"""
    try:
        result = await handle_browse(thread_id, path, extensions)
        return result
    except Exception as e:
        raise HTTPException(status_code=403, detail=str(e))

@router.get("/read")
async def read_file(
    thread_id: str = Query(..., description="Thread ID"),
    path: str = Query(..., description="Virtual file path"),
    start_line: Optional[int] = Query(None, description="Start line number"),
    end_line: Optional[int] = Query(None, description="End line number")
):
    """Read file content"""
    try:
        result = await handle_read_file(thread_id, path, start_line, end_line)
        return result
    except Exception as e:
        raise HTTPException(status_code=403, detail=str(e))

@router.post("/write")
async def write_file(request: WriteFileRequest):
    """Write file content"""
    try:
        result = await handle_write_file(
            request.thread_id,
            request.path,
            request.content,
            request.append
        )
        return result
    except Exception as e:
        raise HTTPException(status_code=403, detail=str(e))

@router.delete("/delete")
async def delete_file(
    thread_id: str = Query(..., description="Thread ID"),
    path: str = Query(..., description="Virtual file path")
):
    """Delete a file"""
    try:
        result = await handle_delete_file(thread_id, path)
        return result
    except Exception as e:
        raise HTTPException(status_code=403, detail=str(e))
```

## 7. 安全要求

### 7.1 路径验证

所有接口必须使用现有的路径验证机制：

```python
# 始终验证虚拟路径
validate_local_tool_path(path, thread_data, read_only=...)

# 始终解析并验证实际路径
actual_path = _resolve_and_validate_user_data_path(path, thread_data)

# 验证后检查路径是否在允许范围内
try:
    Path(actual_path).relative_to(Path(workspace_path).resolve())
except ValueError:
    raise PermissionError("Access denied: path traversal detected")
```

### 7.2 文件大小限制

读取文件时限制最大大小：

```python
MAX_FILE_SIZE = 10 * 1024 * 1024  # 10MB
if os.path.getsize(actual_path) > MAX_FILE_SIZE:
    return {"error": "File too large"}, 400
```

### 7.3 只读路径保护

某些路径只允许读取，不允许写入：

```python
# Skills 路径 - 只读
if _is_skills_path(path) and not read_only:
    raise PermissionError("Write access to skills path is not allowed")

# ACP workspace 路径 - 只读
if _is_acp_workspace_path(path) and not read_only:
    raise PermissionError("Write access to ACP workspace is not allowed")
```

### 7.4 支持的文件类型

前端只显示以下文件类型，但后端不限制读取：
- `.md` - Markdown
- `.txt` - 文本文件
- `.json` - JSON
- `.log` - 日志文件

```python
def get_thread_data(thread_id: str) -> ThreadDataState:
    """Get thread data for a given thread ID"""
    # 方式 1: 从数据库或缓存获取
    thread = get_thread_from_db(thread_id)

    # 方式 2: 构建 thread_data
    workspace_path = get_workspace_path(thread_id)
    uploads_path = get_uploads_path(thread_id)
```

```python
# 限制目录深度
def list_directory(path: str, max_depth: int = 1):

    entries = []
```

## 15. 与前端对接说明

前端将调用以下接口：

```typescript
// 浏览目录（带过滤）
GET /api/filesystem/browse?thread_id={threadId}&path=/mnt/user-data/workspace&extensions=md,txt,json,log

// 读取文件
GET /api/filesystem/read?thread_id={threadId}&path=/mnt/user-data/workspace/file.md

// 保存文件
POST /api/filesystem/write
Body: { 
  thread_id: string,
  path: string,
  content: string
}

// 删除文件
DELETE /api/filesystem/delete?thread_id={threadId}&path=/mnt/user-data/workspace/file.md
```

确保接口路径、参数名、响应格式与前端需求一致。
