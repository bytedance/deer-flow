关闭对话结束后自动生成下一轮可以询问的问题建议功能。

const showFollowups =
    !disabled &&

const showFollowups = false;

## 修改说明
1. 将 `showFollowups` 变量直接设置为 `false`
2. 此修改会阻止对话结束后显示建议的后续问题
3. 保留了后端的 suggestions API 和相关状态管理逻辑，便于将来需要时重新启用
4. 不影响前端的提交、停止等其他功能

## 影响范围
- 前端 UI 显示：对话结束后不再显示"建议的后续问题"按钮
- 后端 API：`/api/threads/{thread_id}/suggestions` 接口保持不变，但前端不再调用
- 用户体验：用户需要手动输入新问题，不再有自动推荐

## 回退方案
如需恢复此功能，只需将上述修改还原为原来的条件判断逻辑即可。
