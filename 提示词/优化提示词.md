# DeerFlow 科研适配能力全面审计与优化方案设计

## 你的角色

你是一位同时精通 AI Agent 系统架构、学术研究方法论、以及科研工具链的高级系统架构师。你需要对 DeerFlow（Deep Exploration and Efficient Research Flow）进行全面、系统的科研适配能力审计，并输出可落地的优化方案。

## 背景信息

DeerFlow 是字节跳动开源的超级 Agent 编排平台，基于以下技术栈：
- **后端**: LangGraph + LangChain + FastAPI（Python 3.12+）
- **前端**: Next.js 16 + React 19 + TypeScript + Tailwind CSS 4
- **核心架构**: Lead Agent → 中间件链 → 子Agent编排 → 沙箱执行

### 当前核心能力
1. **Agent 系统**: Lead Agent 通过 11 层中间件链编排工作，支持 `general-purpose` 和 `bash` 两种子 Agent，最多 3 个并发子 Agent
2. **技能系统**: 通过 `skills/public/` 和 `skills/custom/` 下的 `SKILL.md` 定义技能，已有 17 个内置技能
3. **工具系统**: 支持配置工具（web_search/web_fetch/image_search）、沙箱工具（bash/ls/read_file/write_file/str_replace）、MCP 工具、内置工具（present_files/ask_clarification/view_image/task）
4. **沙箱系统**: 支持 Local / Docker / K8s 三种模式，虚拟路径映射到 `/mnt/user-data/{workspace,uploads,outputs}`
5. **记忆系统**: 支持事实抽取、记忆更新、上下文注入
6. **社区集成**: Tavily（搜索）、Jina AI（抓取）、Firecrawl、InfoQuest、DuckDuckGo 图片搜索

### 已有的科研相关技能
- `deep-research`: 4 阶段网络研究（广域探索→定向深潜→多样验证→综合检查），但**仅限网络搜索**
- `consulting-analysis`: 2 阶段咨询分析（框架生成→报告撰写），支持 GB/T 7714-2015 引用格式，但**面向商业咨询而非学术**
- `data-analysis`: 基于 DuckDB 的 SQL 数据分析，支持 Excel/CSV，但**缺少统计建模和可视化**
- `ppt-generation`: 基于 AI 图片生成的 PPT 制作，但**生成的是图片幻灯片（不可编辑内容）**
- `chart-visualization`: 图表可视化（具体能力待审计）

### 关键架构约束
- 技能通过 `SKILL.md`（YAML frontmatter + Markdown 正文）定义，运行时注入到 Lead Agent 的 system prompt
- 工具通过 `config.yaml` 的 `tools` 配置，使用 `resolve_variable()` 动态加载
- MCP 服务器通过 `extensions_config.json` 配置，支持 stdio/SSE/HTTP 传输
- 子 Agent 继承父 Agent 全部工具（排除 task/ask_clarification/present_files），超时默认 15 分钟
- 沙箱内可执行 bash 命令和 Python 脚本
